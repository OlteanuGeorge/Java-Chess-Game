# Chess Game

## Overview
This Chess Game project is implemented in Java and provides a fully functional chess game with all standard chess rules. It includes features such as move validation, check detection, checkmate detection, pawn promotion, castling, and en passant. 

## Features
- Complete implementation of chess game logic.
- Supports two-player gameplay.
- Move validation: Ensures that players can only make legal moves according to standard chess rules.
- Check detection: Automatically detects when a player's king is in check.
- Checkmate detection: Determines when a player's king is in checkmate, ending the game.
- Pawn promotion: Allows pawns to be promoted to other pieces upon reaching the opponent's back rank.
- Castling: Implements the castling move for kings and rooks.
- En passant: Handles the en passant capture rule for pawns.

## Technologies Used
- Java programming language.
